# Extension Icons

Place your extension icons here:
- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

For the hackathon MVP, you can use placeholder icons or generate simple ones with a shield/eye icon representing "Sentinella" (sentinel/watchful eye).

Suggested design:
- Shield icon with an eye in the center
- Color scheme: Purple/Blue gradient (#6366f1 to #8b5cf6)
- Transparent background
